# 📊 Business Sales Performance Analysis & Dashboard

## 📌 Project Overview
This project focuses on analyzing a **Sales Force dataset** to derive meaningful business insights and build an **interactive Power BI dashboard**.

The goal is to transform raw sales data into actionable insights that support **data-driven decision-making**.

---

## 🎯 Objectives
- Clean and preprocess raw sales data  
- Perform exploratory data analysis (EDA)  
- Build Pivot Tables in Excel for initial insights  
- Develop an interactive Power BI dashboard  
- Generate key business insights and recommendations  

---

## 🗂️ Dataset Information
- **Dataset Name:** Sales Force Dataset (Sample Superstore-like dataset)  
- **Source:** Kaggle  
- **Data Includes:**
  - Order Details (Order ID, Order Date, Ship Date)
  - Customer Information
  - Product Categories & Sub-Categories
  - Sales, Profit, Quantity
  - Region & Location Data  

---

## 🛠️ Tools & Technologies Used
- **Microsoft Excel** → Data Cleaning, Pivot Tables, Initial Analysis  
- **Power BI** → Data Modeling & Dashboard Creation  
- **DAX** → Measures & Calculations  

---

## 🔄 Project Workflow

### 1. Data Collection
- Imported dataset from Kaggle into Excel  
- Verified structure, columns, and data types  

---

### 2. Data Cleaning (ETL Process)

#### 🔹 Handling Missing Values
- Checked for null or blank entries  
- Removed or replaced missing values where necessary  

#### 🔹 Data Formatting
- Converted date columns into proper date format  
- Standardized categorical values (Region, Category, etc.)  

#### 🔹 Removing Duplicates
- Identified duplicate records using Excel tools  
- Ensured data consistency  

#### 🔹 Data Transformation
- Created new calculated columns:
  - Profit Margin  
  - Year / Month from Order Date  

---

### 3. Exploratory Data Analysis (EDA)
Performed using **Excel Pivot Tables**:

- Sales by Region  
- Sales by Category & Sub-Category  
- Monthly Sales Trends  
- Top 10 Products by Sales  
- Profit Analysis  

---

### 4. Excel Analysis (Pivot Tables & Charts)
- Built Pivot Tables for:
  - Region-wise Sales  
  - Date-wise Trends  

- Created:
  - Bar Charts for comparison  
  - Line Charts for trends  

---

## 📈 Power BI Dashboard Development

### 🔹 Data Import
- Loaded cleaned Excel dataset into Power BI  

### 🔹 Data Modeling
- Created relationships (if required)  
- Built DAX measures:
  - Total Sales  
  - Total Profit  
  - Profit Ratio  

---

### 🔹 Dashboard Features

#### 📊 KPIs
- Total Sales  
- Total Profit  

#### 📈 Visualizations
- Sales by Region (Pie Chart)  
- Monthly Sales Trend (Line Chart)  
- Top 10 Products by Sales (Bar Chart)  
- Category-wise Sales Distribution  

#### 🎛️ Filters / Slicers
- Order Date  
- Region  
- Category  

---

## 📊 Key Insights
- West region generated the highest sales revenue  
- Top products contribute a significant portion of total sales  
- Sales fluctuate over time with identifiable trends  
- Some low-sales regions may still be profitable  

---

## 💡 Business Recommendations
- Focus on high-performing regions for growth  
- Improve strategy in low-performing regions  
- Optimize inventory for top-selling products  
- Review underperforming categories  

---

## 📁 Project Deliverables
- Cleaned Excel file with Pivot Tables  
- Power BI Dashboard (.pbix file)  
- Insights & Recommendations summary
